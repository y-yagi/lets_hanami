### commands

```ruby
bundle exec hanami --help
Commands:
  hanami assets [SUBCOMMAND]    # Manage assets
  hanami console                # Starts a hanami console
  hanami db [SUBCOMMAND]        # Manage set of DB operations
  hanami destroy [SUBCOMMAND]   # Destroy hanami classes
  hanami generate [SUBCOMMAND]  # Generate hanami classes
  hanami help [COMMAND]         # Describe available commands or one specific command
  hanami new PROJECT_NAME       # Generate a new hanami project
  hanami rackserver             # [private]
  hanami routes                 # Prints the routes
  hanami server                 # Starts a hanami server
  hanami version                # Prints hanami version
```
